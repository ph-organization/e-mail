# e-mail
### 第三阶段任务

#### 一、短信发送功能

- 要求：用户短信发送(**短信发送不需要实现**)功能,用户可选这只发送短信或邮件,或者两种方式都使用.发送记录可以查询.

#### 二、邮件和短信模板管理

- 管理邮件和短信的发送模板,可以自定义给某些角色发送的短信和邮件的内容.


##### 
















#### 三、 接口权限

- 要求：接口管理功能,对所有已开发的接口管理,用户->角色->接口,进行接口数据管理.哪些用户可以看到使用哪些接口.

#### 四、角色组
- 要求：角色组管理功能,角色组也是一种角色,它可以包含多个角色,也可以直接配置一些接口权限.
